# CrashBox 手动测试文档

## 测试环境

- 操作系统: Windows 11
- 编译器: GCC 15.2 (MinGW-w64)
- 图形库: SFML 3.0.1
- 分辨率: 1280x720

## 测试项目

### 1. 启动和初始化

| 测试项 | 预期结果 | 验证方法 |
|--------|----------|----------|
| 程序启动 | 窗口正常打开，显示标题"CrashBox - 2D车辆物理模拟器" | 运行CrashBox.exe |
| 配置加载 | 控制台输出"已加载配置文件" | 查看控制台/日志 |
| 字体加载 | HUD中文文字正确显示 | 观察左上角HUD |
| 地形生成 | 绿色折线地形可见 | 观察下方地形 |
| 车辆初始化 | 弹簧网络和轮子可见 | 观察车辆区域 |

### 2. 车辆物理

| 测试项 | 预期结果 | 验证方法 |
|--------|----------|----------|
| 重力落地 | 车辆启动时轻微落地弹跳后稳定 | 观察启动后行为 |
| 加速 | 按W后速度增加，HUD数值上升 | 按住W观察速度 |
| 刹车 | 按S后速度下降 | 加速后按S |
| 制动重心转移 | 刹车时观察前悬挂压缩增加 | 高速行驶后急刹 |
| 坡道行驶 | 车辆在上坡减速，下坡加速 | 驾驶到坡道区域 |

### 3. 悬挂系统

| 测试项 | 预期结果 | 验证方法 |
|--------|----------|----------|
| 悬挂压缩 | HUD显示压缩百分比在0-100%之间 | 正常行驶观察 |
| 悬挂回弹 | 经过凹坑后悬挂有回弹动作 | 驾驶到凹坑区域 |
| 悬挂断裂 | 高速撞击后悬挂显示[断裂] | 高速冲向地形台阶 |

### 4. 碰撞与损坏

### 5. 地形系统

## 已知限制

1. 2D侧视图模拟，转向为水平移动而非实际转向
2. 控制台日志中文在Windows控制台可能显示乱码（窗口内正常）
3. 极端碰撞偶尔可能出现物理不稳定（弹簧力过大导致振荡）

## 测试通过标准

- 所有基础功能测试项通过
- 连续运行5分钟无崩溃
- 窗口内中文HUD正确显示
- 键盘控制响应灵敏
